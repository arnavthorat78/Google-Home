Links

See the keyboard shortcuts below for quickly navigating through popular links.

Keyboard Shortcut Definition
Ctrl + Alt + H Open up the Home page.
Ctrl + Alt + S Open up the Search page.
Ctrl + Alt + W Open up the Weather page.
Ctrl + Alt + G Open up the Settings page.
Ctrl + Alt + F Open up the Feedback page.
